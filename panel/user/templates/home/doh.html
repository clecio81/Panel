Configurar DoH no seu navegador

Existem vários navegadores compatíveis com DNS sobre HTTPS (DoH). Este protocolo permite você criptografa sua conexão para 1.1.1.1 para proteger suas consultas de DNS contra invasões de privacidade e adulteração. Alguns navegadores podem já ter esta configuração habilitada.
Mozilla Firefox
Selecione o botão menu > Configurações.No menu Geral, role para baixo para acessar Configurações de rede. Selecione Configurações . Selecione Ativar DNS sobre HTTPS.
Google Chrome
Selecione o menu três pontos em seu navegador > Configurações. Selecione Privacidade e segurança > Segurança. Role para baixo e ative Usar DNS seguro.
Microsoft Edge
Selecione o menu três pontos em seu navegador > Configurações. Selecione Privacidade, Pesquisa e Serviços e role para baixo até Segurança. Ativar Usar DNS seguro. Selecione Escolher um provedor de serviços.
